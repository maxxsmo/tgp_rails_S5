# README

## projet :

Première appli en Rails qui reprend le concept d'application de The Gossip Project.

## fonctionnement 

- Cloner le repository + lancer l'appli
- `$ rails db:migrate`
- `$ rails db:seed`
- `$ rails server`
- go à l'URL :localhost:3000/home
- enjoy ;)
